# Usage
``` shell
# Just Patch
python patch.py [boot.img]
# Use Custom Magisk.apk to Patch
python patch.py [boot.img] [Magisk.apk]
# Use Custom Magisk.apk to Patch with Arch
python patch.py [boot.img] [Magisk.apk] [ARCH]
```
